Orchestroids

About
Orchestroids is a 2d shooting game made with 3d models that focuses on helping people with musical tone recognition. Asteroids come from outer space and are headed towards planet earth! The players job is to use a sound frequency cannon to destroy the asteroids. Each astroid has a sequence of musical notes that your cannon has to match. The more you play the better you become at identifying the notes.

Comments
This was made for a game programming class I took in 2015. The task of this project was to create a game that would teach something specific. The game was developed with one other person and was the first time I used unity and 3d modeling software. This project helped me understand object oriented programming and visually showed me how important clean and efficient code is.

    Features
  • Tutorial
  • Change difficulty
  • Load/Reset/Fire cannon
  • High scores
  • Keyboard controller
  • Changing scenes
  • Sounds/Music
  • Animations
    Tools Used
  • Unity Engine
  • Visual studios
  • C#
  • JavaScript
  • Blender

Wizdom

About
Wizdom seeks to connect the youth and seniors of Roosevelt Island by creating a platform where both age groups can connect and share valuable life skills. Ultimately, wizdom hopes to create a bridge between the different generations that was not as prevalent before, a bond that goes beyond a mentor-mentee experience and a bond that can eventually turn into a friendship.
